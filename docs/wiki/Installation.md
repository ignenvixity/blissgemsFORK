# Installation

## Requirements

- Paper/Spigot server on Minecraft `1.21.x`
- Java `21` runtime recommended

## Install Steps

1. Download `BlissGemsFORKED-1.0.jar` from [Releases](../../releases).
2. Stop your server.
3. Put the jar into your server `plugins/` folder.
4. Start the server once to generate plugin files.
5. Edit `plugins/BlissGems/config.yml`.
6. Restart the server or run `/bliss reload`.

## Update Steps

1. Back up your existing `plugins/BlissGems/` folder.
2. Replace the old jar with the new one.
3. Start server and review console for migration/config messages.
4. Compare new default config keys if needed.

## Verify Plugin Loaded

- Run `/bliss` in game.
- Confirm no startup errors in console.
- Confirm expected commands/permissions are registered.
